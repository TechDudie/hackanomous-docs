# Free Resources for Hack Clubbers

Hack Club provides a number of free resources so you can spend more time building and less time paying for services.

As a Hack Club member, you have access to:

- Free AI credits
- Free Linux hosting
- A free custom domain
- A free global CDN

These tools are perfect for building and shipping your Hackanomous projects.

> **Good news**
>
> Every resource listed below is completely free and **does not require a credit card**.

---

# Free AI Credits

Use **Hack Club AI** in the projects, tools and agents that **you build yourself**.

It provides free access to many of today's leading AI models, allowing your applications to use large language models without paying for API keys.

## Hack Club AI

**Website**

```text
https://ai.hackclub.com
```

### What you get

- Free access to frontier AI models
- Gemini
- GPT
- Kimi
- Qwen
- DeepSeek
- 30+ additional models

Hack Club AI is intended for software, tools and AI agents that **you build yourself**.

> **Important**
>
> Please do **not** connect Hack Club AI to off-the-shelf coding assistants such as Cursor, OpenClaw, or similar tools.
>
> Likewise, don't use it to automatically generate your entire project. The goal of Hackanomous is to help you learn by building.

# Free Hosting, Domains and a CDN

Once you've built your project, you'll need somewhere to host it.

Hack Club offers three excellent free services:

| Service | Purpose |
|---------|---------|
| **Nest** | Free Linux VPS hosting |
| **Free Domains** | A free `.dino.icu` subdomain |
| **CDN** | Global hosting for images, videos and other assets |

---

# Nest

Run almost anything on your own Linux virtual server.

Nest is ideal for:

- Websites
- APIs
- Discord or Slack bots
- Databases
- Web servers
- Custom applications

**Website**

```text
https://hackclub.app
```

---

# Free Domains

Claim your own custom subdomain ending in:

```text
.dino.icu
```

Getting one simply involves opening a Pull Request to the Hack Club DNS repository.

If you've never done that before, don't worry—we have a full guide explaining every step.

**Repository**

```text
https://github.com/hackclub/dns
```

---

# CDN

Need somewhere to host images, videos or downloads?

Hack Club's CDN lets you upload files and receive a fast global URL that can be used anywhere in your projects.

Perfect for:

- Images
- Videos
- PDFs
- Downloads
- Static assets

**Website**

```text
https://cdn.hackclub.com
```

---

> **New to Pull Requests?**
>
> Claiming a `.dino.icu` domain requires opening a GitHub Pull Request.
>
> It might sound intimidating at first, but it's actually very straightforward.
>
> Follow the **Domain Guide** for a complete walkthrough, and if you get stuck, ask in **#hackanomous-help** on the Hack Club Slack.
